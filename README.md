# 课程作业
借鉴了**[GNMM-mf](https://github.com/GNMM-mf/Piecewise-Trajectory-Optimization)**的工作

## 将代码克隆到ROS工作空间

## 编译 刷新
```
catkin_make
source devel/setup.sh
```
## 运行
```
roslaunch astar_path_planner astar_planner.launch
```
新建终端
```
source devel/setup.sh
roslaunch astar_path_planner global_trajopt.launch
```
